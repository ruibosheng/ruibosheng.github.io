基于hexo搭建个人博客-建站篇（win10）

发表于 2022-04-21 更新于 2022-05-10

前言

正如某博主所说”对任何渴望进步的人来说，写博客/文章/回答对自己的成长帮助都是巨大的。频率和数量不必过分强求，有话要说就写写，没事就歇歇，随意点。因为坚持写作的那一刻起，你就已经开始受益。”搭建博客花了好几天，期间踩坑无数，遂记录下来以供后来者参考。

准备工作

  • 下载安装node.js https://nodejs.org/en/

  • 键盘win+r，弹出运行框输入cmd，进入如下界面，分别输入node -v和npm -v，回车，显示版本号说明安装成功

  • 下载安装git https://git-scm.com/downloads

  • 安装后桌面右键会出现下图

  • git安装成功后需要配置邮箱、姓名和SSH密钥，网上有很多教程，在此不展开描述了，想要了解更多，详见廖雪峰git教程 https://www.liaoxuefeng.com/

  • 注册github https://github.com/

  • 注册后直接登录

  • 新建一个仓库

  • 创建仓库名：用户名小写.github.io，这就是博客创建成功后的访问地址，现在还不能用，创建好先放在这里。

    npm install -g cnpm –registy=http://registry.npm.taobao.org

  • 自己挑一个盘(比如G盘)，命令行输入G:，再输入mkdir blog，打开G盘发现多了一个blog文件夹，回到命令行输入cd blog进入blog目录，输入代码hexo init，等几分钟

  • 这里git必须是配置好邮箱和用户名，否则无法执行hexo init
  • 下图说明本地hexo博客已经生成

  • 输入hexo s，复制生成的地址转到浏览器粘贴打开，即可预览博客
